# Python Setup & Run Guide

## Run Python File

```bash
cd folder
python3 main.py
```

## Create Virtual Environment

```bash
python3 -m venv myenv
```

## Activate Virtual Environment

```bash
source myenv/bin/activate
```

## Install Packages

```bash
pip install pandas
```

## Notes

- Use `python3` instead of `python`
- Use virtual environments for all projects
- Run scripts from the correct directory or use full path
